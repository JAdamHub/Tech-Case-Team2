Document Upload

To upload markdown (.md) files, please use GitHub's web interface:

Files should be uploaded to: converter_markdown/input_folder_markdown/

Only .md files are allowed

Upload Files on GitHub

Instructions:

  1. Click the "Upload Files on GitHub" button above
  2. Drag and drop your .md files or click to select them
  3. Add a commit message (e.g., "Add markdown files")
  4. Click "Commit changes"
