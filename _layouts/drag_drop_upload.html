Document Upload

Upload Instructions

  1. Click the button below to open GitHub's upload interface
  2. Drag and drop your .md files or click to select them
  3. Make sure files are uploaded to: converter_markdown/input_folder_markdown/
  4. Add a commit message (e.g., "Add markdown files")
  5. Click "Commit changes"
  6. The files will be automatically processed and added to the repository
